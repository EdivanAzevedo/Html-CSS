🤓 Como utilizar audio 🔊

Clique AQUI para voltar a página inicial

Para adicionar audios, basta utilizar a tag audio, para o audio tocar sozinho, deve-se utilizar o atributo autoplay, e para aparecer o controle de audio, utiliza o atributo controls

Ou então você pode utilizar a tag audio, e dentro dela utilizar a tag

Vale ressaltar que dentro da tag audio você pode utilizar o atributo preload, de modo que a página só irá carregar depois que exercutar o que estiver ali.

  • auto = a página só irá carregar depois que todo o arquivo de áudio carregar
  • metadata = a página só irá carregar depois que carregar alguns dados do arquivo de áudio
  • none = a página irá carregar independente de conseguir ou não carregar o arquivo
  • loop = ao terminar de executar o arquivo, ele irá carrega-lo novamente
