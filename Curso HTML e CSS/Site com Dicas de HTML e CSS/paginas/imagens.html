A imagem original era muito grande, para deixar do tamanho das imagens acima, utilizei os atributos width e height informando a quantidade de pixels que eu quero utilizar na imagem.

Imagens dinâmicas

Para criar imagens dinâmicas, utiliza a tag picture, esta tag permite você adicionar uma imagem dentro do corpo dela, contudo permite você adicionar várias source media type, (camadas) para deixar a imagem dinâmica.
