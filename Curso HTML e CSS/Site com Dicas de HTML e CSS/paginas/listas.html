Listas ordenadas

Para criar uma crista ordenada utiliza a tag ol, e dentro do scopo deste tag se utiliza a tag li para definir cada elemento da linha, também se pode utilizar o atributo type para definir o tipo de numeração da lista, e utilizar a tag start para definir a partir de qual ponto começa a lista exemplo:

  1. Posição 1
  2. Posição 2
  3. Posição 3

Lista não ordenadas

São listas que você não precisa enumerar, e sim somente manter listar, utiliza-se a tag ul para tal feito, exemplo:

  • Posição 1
  • Posição 2
  • Posição 3
