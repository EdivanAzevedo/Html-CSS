Quando utilizar seletores?

Seletores são uteis quando se quer dar funções a elementos diferentes no desenvolvimento em HTML

Existe alguns tipos de seletores com funções diferentes, estes tipos são:

    Seletores Personalizados

  • # = id

  • . = class

  • : = pseudo-class

  • :: = pseudo-element

  • > = children
