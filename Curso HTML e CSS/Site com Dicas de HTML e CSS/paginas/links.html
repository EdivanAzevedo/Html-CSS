Links externos

Para criar um texto que ao clicar se direciona para outra página, deve-se ancorar o texto, utilizando a tag a.

Também pode utilizar os atributos targed e rel dentro da tag para ela abrir em uma nova aba, como no exemplo abaixo:

A palavra YouTube é uma ancôra para o site do YouTube, contudo somente esta palavra do YouTube irá abrir uma nova aba ao clicar.

Vale ressaltar que ao utilizar o atributo rel, serve para dizer a algum mecânismo de busca, o que este link está fazendo, quanl a relaçao dele com a minha página.

Links internos

Para criar um texto que ao clicar te direciona para uma outra página do mesmo site (este), eu devo utilizar como parâmentro em href o nome da minha segunda página.

Ao clicar aqui,você será direcionado para uma segunda página deste site.

Quando eu não quiser que o mecânismo de busca siga o link, eu utilizo no atributo rel o termo: nofollow

Links para download

Para criar um link para download, deve-se utilizar a tag a para obter o link, e dentro desta tag a, deve-se utilizar os atributos download e o type, sendo que o type deve-se ter como parâmentro application/alguma coisa. Exemplo:

Ao clicar aqui você irá baixar meu WEBIBLIOMINING em tipo pdf

Ao clicar aqui você irá baixar meu WEBIBLIOMINING em tipo rar
