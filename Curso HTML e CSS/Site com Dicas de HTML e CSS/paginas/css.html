Cascading Style Sheets é um mecanismo para adicionar estilos a uma página web, aplicado diretamente nas tags HTML ou ficar contido dentro das tags style. Também é possível, adicionar estilos adicionando um link para um arquivo CSS que contém os estilos.

Não tem muito segredos para sua utilização, já que sua única função é deixar a página bonita

    Algumas funções comum em CSS

  • color = definir a cor
  • font-family = definir a fonte
  • font-size = definir tamanho da fonte
  • text-align = definir alinhamento do texto
  • text-decoration = definir decoração na fonte
  • background-image = definir imagem como backgroud
  • background-color = definir uma cor como backgroud
  • rgb = padrão de cores
  • rgba = padrão de cores e definição de transparencia
  • border-radius = arredondamento de borda
  • padding-left = movimentar à esquerda
  • margin= utilizado para definir espaço vazio ao redor de um elemento
  • list-style-type = quando utilizado com none remove os pontos das listas
  • filter = utilizado para definir filtros para uma imagem
  • background-image: linear-gradient() = Após por a direção e as cores utilizadas, irá formar um degradê
  • background-attachment = atribui status ao background
  • * = utiliza antes de aspas para criar configurações globais
