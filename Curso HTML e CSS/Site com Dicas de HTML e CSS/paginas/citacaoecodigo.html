Citações

Posso utilizar a tag q para criar uma citação, ou seja, deixar entre aspas, exemplo:
Este trecho está citado

Também posso utilizar blockquote, juntamente com cite dentro de sua tag, onde o blockquote faz um bloco com a citação enquanto o cite ajuda os mecanismos de busca identificar a citação e referênciar o livro original exemplo:

A internet surgiu como um fenômeno nos últimos tempos, que veio para agregar conhecimento, interatividade, entreterimento e principalmente informação para as pessoas.
