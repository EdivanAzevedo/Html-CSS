Aqui veremos como inserir vídeos hospedados localmente no site

Para adicionar um vídeo. basta utilizar a tag video, você pode utilizar o atributo height para diminuir o tamanho do vídeo, você também pode utilizar o atributo controls para exibir o menu de controle no vídeo. Para adicionar uma thumbnail, você utilizar o atributo poster

Contudo tal método não é eficiente, por conta do alto consumo de dados do seu servidor, para solucionar tal problema, o melhor modo de adicionar um vídeo, é adionando um vídeo externo.
